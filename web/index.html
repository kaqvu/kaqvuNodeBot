Komendy zarządzania:
.create <nazwa> <ip:port> <wersja>
.create .randomname <ip:port> <wersja> [liczba]
.start <nazwa|*> [flagi]
.stop <nazwa|*>
.delete <nazwa>
.logs <nazwa|*>
.list
.clear
.blockmessages
.blockcommands
.help

Komendy akcji:

.loopuse <nazwa|*>
.walk <nazwa|*> <forward|back|left|right>
.dropitem <nazwa|*> <slot>
.look <nazwa|*> <yaw|kierunek> [pitch]
.setslot <nazwa|*> <0-8>
.rightclick <nazwa|*>
.leftclick <nazwa|*>
.guiclick <nazwa|*> <0-53>
.autoeat <nazwa|*>
.follow <nazwa|*> <nick>
.autofish <nazwa|*>
.goto <nazwa|*> <x> <y> <z>
.attack <nazwa|*> <mob|player|all> <range>
.stats <nazwa|*>
.eqclick <nazwa|*> <slot> <button> <mode>

-joinsend <wiadomosc>
-reconnect

-jumpafk

-sneakafk:<sekundy>

-delayflag <ms>
-setslot <0-8>

Kierunki dla .look:
north, south, east, west, up, down

Przykłady:
.create bot1 hypixel.net 1.8.9
.start bot1 -reconnect -jumpafk
.logs bot1
.look bot1 north
.goto bot1 100 64 200
.eqclick bot1 38 0 0
.eqclick bot1 38 1 0
